Detailed Description

Represents a Corintho game state.

Note
The class is designed to be as memory efficient as possible since it is contained in each node of the Monte Carlo Search Tree which is the main memory bottleneck of the program
The game state does not know if it is a terminal state This is deduced by the Node class from the return value of getLegalMoves (when there are no legal moves)
